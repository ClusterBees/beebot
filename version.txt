🍯 **v0.2.2 — Cozy DMs & Empowered Commands**  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**💌 Direct Message Responsiveness**  
• BeeBot now lovingly responds to DMs with emotion-aware threading and adaptive tone matching.  
• Private chats are now safe havens, gated by user consent and personalized comfort.

**⚙️ New Slash Commands**  
• `/dm` delivers personalized, affirming messages directly to your inbox.  
• `/invite` creates dynamic server invites using the app’s client ID.  
• `/bee_name` auto-applies a custom bee name as your nickname (when allowed).

**🔐 Consent & Context Enhancements**  
• `on_message()` now handles DMs smoothly, logging context with precision and privacy (`dm:{user_id}`).  
• Unified command processing makes BeeBot more reliable across all channels.

**📝 Refactored Command List**  
• Full command library updated to reflect latest emotional tools and support options.  
• Nickname changes and DM handling now feature friendly ephemeral error messages.
